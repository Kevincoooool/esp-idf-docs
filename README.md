## 注意事项

1. Korvo / P4 **例程 bin 不可互烧**；`idf.py set-target` 与板子一致  
2. 换工程必须 `cd` 到新目录；大改配置可删 `build/`  
3. 首次编译需联网（`managed_components`）  
4. P4C5 Wi-Fi 需等 C5 就绪（约 10～20 s）
