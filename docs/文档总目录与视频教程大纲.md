# 文档总目录与视频教程大纲

> **用途**：本仓库全部 Markdown 的 **唯一总索引**；按章节顺序录制即可得到完整视频课。  
> **维护**：新增/改文档时同步更新本页集号与链接。  
> **例程资料**：淘宝酷世DIY 购板 → 联系客服 → **百度网盘**（见 [README](../README.md)）。

---

## 一、文档四层体系

```text
第 0 层 · 总控（你现在看的）
    └── 文档总目录与视频教程大纲.md

第 1 层 · 入门与框架（所有观众先看）
    └── 通用-搭建编译环境 / 框架构成 / 工程结构 / 例程开发流程 …

第 2 层 · API 概念（配合例程穿插讲）
    └── 通用-gpio / wifi / audio / lvgl …（旧版笔记标「附录」）

第 3 层 · 板级路线（二选一，不要混烧）
    ├── Korvo：开发板介绍 → 学习指南 → Korvo例程详解-*
    └── P4C5：开发板介绍 → 学习指南 → P4C5例程详解-*

第 4 层 · 专题与查表
    └── 对照表 / 05.test / GMF / SP V4 / 在线组件
```

| 前缀 | 数量级 | 视频用法 |
|------|--------|----------|
| `通用-` | ~35 篇 | **概念课** + 屏幕录 IDE/API |
| `Korvo例程详解-` | 15 篇 | **实操课**，必须 Korvo 板 |
| `P4C5例程详解-` | 15 篇 | **实操课**，必须 P4C5 板 |
| 板介绍/学习指南 | 各 2～3 篇 | **系列开篇 / 路线图** |
