## 仅一块板有的能力（勿搞混）

| 能力 | 板子 | 说明 |
|------|------|------|
| DVP 摄像头 + 网页 `/stream` MJPEG | **Korvo** | P4 走 MIPI CSI / UVC |
| BluFi 蓝牙配网例程 | **Korvo** | P4 小智工程内触摸配网 |
| ADC 六键 | **Korvo** | P4 为 GPIO35 / GPIO0 |
| 小智 / esp-album / GMF / 05.test | **P4C5** | Korvo 包内无对等同名工程 |
| 硬件 JPEG + PPA 解 AVI | **P4C5** | Korvo AVI 为软件解码 |
| Wi-Fi 内置 S3 | **Korvo** | P4 需等 C5 协处理器 |
